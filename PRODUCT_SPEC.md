# Personal Website v1

## Goal
A dark, terminal-inspired personal website that combines portfolio, technical writing, and a clean markdown writing studio.

## Audience
Recruiters, engineers, collaborators, and technically inclined readers.

## Core Sections
- Home / About
- Projects
- Writing Archive
- Writing Detail
- Resume
- Studio

## Studio Purpose
The Studio is a clean markdown-based writing workspace with:
- markdown input
- live preview
- syntax-highlighted code blocks
- minimal distraction-free interface

## Not in v1
- authentication
- backend persistence
- CMS
- collaborative editing
- comments
- real publishing workflow

## Design Direction
- dark mode by default
- terminal-inspired visual identity
- monospace-forward typography
- minimal but polished interface
- subtle accent color
- strong readability for long-form writing

## Technical Stack
- Next.js
- TypeScript
- Tailwind CSS
- MDX for public writing content

## Requirements
- accessible
- mobile responsive
- keyboard-friendly
- reusable components
- content-driven architecture
- maintainable codebase

## Content Types
### Profile
name, role, bio, avatar, links

### Experience
company, title, period, description

### Project
title, summary, technologies, githubUrl, liveUrl, featured

### Writing
title, slug, date, tags, excerpt, body

## Studio Requirements
- split-pane layout on desktop
- stacked or tabbed layout on mobile
- markdown preview with syntax highlighting
- clean editor surface
- no backend dependency in v1
